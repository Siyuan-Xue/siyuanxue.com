AI 定价这件事，有一种不动声色的聪明。

同样一件事——用量——在消费者和企业市场里，感受完全不同。

对消费者来说，订阅制加用量上限，把「使用」变成了一条价值进度条。

用得越多，越觉得钱花得值。甚至连触顶都会有点像达成成就。然后额度重置，新一轮又开始了。

但在 API 按量付费的模式下，每一次调用都像一笔成本。人的本能会变成少用 token、多做缓存、少调用、优化支出。

到了企业这里，逻辑又反转一次。

API 账单上涨，往往意味着产品使用量增长、客户变多、营收上升。它成了 COGS（销售成本），不再只是「花掉的钱」。

于是，同一个 token 变成两种完全不同的产品：

- 对消费者，它是**订阅权益**。
- 对公司，它是**生产投入**。

这已经不只是定价，而是非常聪明的心理记账。
